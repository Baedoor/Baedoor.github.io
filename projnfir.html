Nimfire

Latest version: 0.1.3

Source code: GitHub

Nimfire is 2D game and GUI library written in Nim, aimed for development of small games. It's very much unfinished and shouldn't be used besides experimenting or projects that are good with rewriting a lot of code in longer term.

Despite being in so early development stage, there is project using Nimfire already:

Installation

Install Nim language (version 2.0 or higher is required). After doing that, run this command in your terminal: nimble install https://github.com/Toma400/Nimfire
This will install Nimfire as a library through Nimble. For further instructions, refer to GitHub guide.
