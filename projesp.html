Espirit

Not released yet

Source code: GitHub

Espirit is parser library for .esp/.esm files used by The Elder Scrolls III: Morrowind, written in Nim. It is in early development stages and has no releases yet - you can still download its HEAD through Nimble, as with any other Nim library.

Installation

Install Nim language. After doing that, run this command in your terminal: nimble install https://github.com/Toma400/Espirit
This will install Espirit as a library through Nimble. For further instructions, refer to GitHub guide.
